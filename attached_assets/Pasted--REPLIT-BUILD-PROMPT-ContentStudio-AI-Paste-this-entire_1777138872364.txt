## BACKEND (Node.js + Express)

### File: `server/index.js`

Create an Express server with these endpoints:

#### POST `/api/generate-story`
- Input: `{ brief: string, genre: string, duration: number }`
- System prompt instructs Claude to return a structured story JSON
- Response: `{ title, synopsis, acts: [{actNumber, title, description, keyMoment}], characters: [{name, description}], mood, colorPalette, musicSuggestion }`

#### POST `/api/generate-video-prompts`
- Input: `{ story: object, style: string, duration: number, part: number, totalParts: number, previousLastFrame: string }`
- Uses the full Seedance 2.0 skill system prompt (embedded below)
- Returns structured shot-by-shot prompts with continuation logic
- Response: `{ shots: [], effectsInventory: [], densityMap: [], energyArc: string, lastFrameDescription: string, copyablePrompt: string }`

#### POST `/api/continue-story`
- Input: `{ existingStory: object, direction: string }`
- Extends the story with new acts
- Response: same story structure with additional acts

#### GET `/api/health`
- Returns `{ status: "ok", timestamp: Date.now() }`

### Claude API Integration (server-side)

Use `@anthropic-ai/sdk` npm package.

```javascript
import Anthropic from '@anthropic-ai/sdk';
const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });
```

**For all API calls use:** `claude-opus-4-5` model, `max_tokens: 4096`

**Embedded Seedance 2.0 System Prompt** (use this exactly in `/api/generate-video-prompts`):

```
You are a specialist AI video prompt writer for Seedance 2.0. Your job is to take a creative brief and transform it into a detailed, shot-by-shot video generation prompt.

CRITICAL RULES:
1. Always output ALL FOUR sections: SHOT-BY-SHOT EFFECTS TIMELINE, MASTER EFFECTS INVENTORY, EFFECTS DENSITY MAP, ENERGY ARC
2. Each shot = 1–4 seconds. Name effects precisely: "speed ramp (deceleration)" not "speed ramp"
3. If 3 effects happen simultaneously, list all 3 explicitly
4. Mark the most impactful shot: "This is the SIGNATURE VISUAL EFFECT"
5. Be specific about speed: "approximately 20–25% speed" not "slow motion"
6. LAST FRAME RULE: Always end with a "CONTINUATION FRAME" section describing exactly what the last frame looks like — subject position, camera angle, lighting, environment state — so the next video part can seamlessly continue.
7. Never let energy drop without intention. Every transition is a creative decision.
8. Return response as valid JSON with this structure:
{
  "shots": [
    {
      "shotNumber": 1,
      "timestamp": "00:00-00:03",
      "name": "Shot name",
      "effects": ["effect1", "effect2"],
      "description": "Visual description",
      "cameraWork": "Camera behaviour",
      "speed": "Speed/timing info",
      "transition": "How this exits to next shot",
      "isSignature": false
    }
  ],
  "effectsInventory": [
    {
      "name": "Effect name",
      "usedCount": 2,
      "shots": [1, 3],
      "role": "Role in edit"
    }
  ],
  "densityMap": [
    {
      "timeRange": "00:00-00:03",
      "density": "HIGH",
      "effects": ["effect1"],
      "count": 3,
      "duration": "3s"
    }
  ],
  "energyArc": {
    "act1": "Description",
    "act2": "Description",
    "act3": "Description"
  },
  "lastFrameDescription": "Exact description of the final frame for seamless continuation",
  "copyablePrompt": "The full plain-text prompt ready to paste into Seedance 2.0"
}
```

STYLE INSTRUCTIONS WILL BE PROVIDED PER REQUEST. Follow them exactly.
```

## COPY PROMPT FEATURE

This is the most critical UX feature. Make it bulletproof:

```javascript
const copyPrompt = async (text) => {
  try {
    await navigator.clipboard.writeText(text);
    setCopied(true);
    setTimeout(() => setCopied(false), 2000);
  } catch {
    // Fallback for non-HTTPS
    const ta = document.createElement('textarea');
    ta.value = text;
    document.body.appendChild(ta);
    ta.select();
    document.execCommand('copy');
    document.body.removeChild(ta);
    setCopied(true);
    setTimeout(() => setCopied(false), 2000);
  }
};
```

Button states:
- Default: "Copy Prompt" with copy icon
- Copied: "Copied!" with checkmark, accent background
- Transition: smooth 150ms color change
