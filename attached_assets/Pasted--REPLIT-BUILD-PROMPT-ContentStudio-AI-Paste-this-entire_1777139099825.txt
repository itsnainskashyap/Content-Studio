## PROJECT OVERVIEW

Build a full-stack web application called **ContentStudio AI** — a professional video prompt generation platform for AI video tools like Seedance 2.0. This is built for serious content creators, filmmakers, and creative directors. It must look like it was designed by a world-class product design team with 20+ years of experience. Zero AI-generated look. No purple gradients. No generic layouts.

**Stack:** React (Vite) + Node.js/Express backend + Claude API (already available in Replit via environment variable `ANTHROPIC_API_KEY`)

## BACKEND (Node.js + Express)

### File: `server/index.js`

Create an Express server with these endpoints:

#### POST `/api/generate-story`
- Input: `{ brief: string, genre: string, duration: number }`
- System prompt instructs Claude to return a structured story JSON
- Response: `{ title, synopsis, acts: [{actNumber, title, description, keyMoment}], characters: [{name, description}], mood, colorPalette, musicSuggestion }`

#### POST `/api/generate-video-prompts`
- Input: `{ story: object, style: string, duration: number, part: number, totalParts: number, previousLastFrame: string }`
- Uses the full Seedance 2.0 skill system prompt (embedded below)
- Returns structured shot-by-shot prompts with continuation logic
- Response: `{ shots: [], effectsInventory: [], densityMap: [], energyArc: string, lastFrameDescription: string, copyablePrompt: string }`

#### POST `/api/continue-story`
- Input: `{ existingStory: object, direction: string }`
- Extends the story with new acts
- Response: same story structure with additional acts

#### POST `/api/generate-music-brief`
- Input: `{ story: object, style: string, mood: string, duration: number, language: string }`
- Claude generates a detailed music brief/prompt for AI music generators (Suno, Udio, etc.)
- Response:
```json
{
  "genre": "string",
  "subGenre": "string",
  "tempo": "string (e.g. 120 BPM, medium-fast)",
  "energy": "string (low/medium/high/explosive)",
  "instruments": ["instrument1", "instrument2"],
  "mood": "string",
  "vocalStyle": "string or null",
  "referenceArtists": ["Artist1", "Artist2"],
  "sunoPrompt": "Ready-to-paste prompt for Suno AI",
  "udioPrompt": "Ready-to-paste prompt for Udio AI",
  "timingNotes": "How music should sync with video parts",
  "partBreakdown": [
    { "part": 1, "musicDirection": "description of how music should feel in this part" }
  ]
}
```

#### POST `/api/generate-voiceover`
- Input: `{ story: object, style: string, language: string, tone: string, duration: number, part: number }`
- Claude writes the actual voiceover script for that video part
- `language` can be: `"hindi"`, `"english"`, `"hinglish"`
- Response:
```json
{
  "language": "hinglish",
  "script": "Full voiceover text",
  "wordCount": 45,
  "estimatedDuration": "12 seconds",
  "tone": "energetic",
  "deliveryNotes": "Speak fast on first line, pause before last line",
  "emphasisWords": ["word1", "word2"],
  "alternateVersions": [
    { "label": "More Dramatic", "script": "..." },
    { "label": "Casual/Conversational", "script": "..." }
  ],
  "elevenlabsPrompt": "Voice style description for ElevenLabs",
  "copyableScript": "Clean plain text for copy-paste"
}
```

#### GET `/api/health`
- Returns `{ status: "ok", timestamp: Date.now() }`

### Claude API Integration (server-side)

Use `@anthropic-ai/sdk` npm package.

```javascript
import Anthropic from '@anthropic-ai/sdk';
const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });
```

**For all API calls use:** `claude-opus-4-5` model, `max_tokens: 4096`

**Embedded Seedance 2.0 System Prompt** (use this exactly in `/api/generate-video-prompts`):

```
You are a specialist AI video prompt writer for Seedance 2.0. Your job is to take a creative brief and transform it into a detailed, shot-by-shot video generation prompt.

CRITICAL RULES:
1. Always output ALL FOUR sections: SHOT-BY-SHOT EFFECTS TIMELINE, MASTER EFFECTS INVENTORY, EFFECTS DENSITY MAP, ENERGY ARC
2. Each shot = 1–4 seconds. Name effects precisely: "speed ramp (deceleration)" not "speed ramp"
3. If 3 effects happen simultaneously, list all 3 explicitly
4. Mark the most impactful shot: "This is the SIGNATURE VISUAL EFFECT"
5. Be specific about speed: "approximately 20–25% speed" not "slow motion"
6. LAST FRAME RULE: Always end with a "CONTINUATION FRAME" section describing exactly what the last frame looks like — subject position, camera angle, lighting, environment state — so the next video part can seamlessly continue.
7. Never let energy drop without intention. Every transition is a creative decision.
8. Return response as valid JSON with this structure:
{
  "shots": [
    {
      "shotNumber": 1,
      "timestamp": "00:00-00:03",
      "name": "Shot name",
      "effects": ["effect1", "effect2"],
      "description": "Visual description",
      "cameraWork": "Camera behaviour",
      "speed": "Speed/timing info",
      "transition": "How this exits to next shot",
      "isSignature": false
    }
  ],
  "effectsInventory": [
    {
      "name": "Effect name",
      "usedCount": 2,
      "shots": [1, 3],
      "role": "Role in edit"
    }
  ],
  "densityMap": [
    {
      "timeRange": "00:00-00:03",
      "density": "HIGH",
      "effects": ["effect1"],
      "count": 3,
      "duration": "3s"
    }
  ],
  "energyArc": {
    "act1": "Description",
    "act2": "Description",
    "act3": "Description"
  },
  "lastFrameDescription": "Exact description of the final frame for seamless continuation",
  "copyablePrompt": "The full plain-text prompt ready to paste into Seedance 2.0"
}
```

STYLE INSTRUCTIONS WILL BE PROVIDED PER REQUEST. Follow them exactly.
```

**Music Brief System Prompt** (use this in `/api/generate-music-brief`):

```
You are a professional music supervisor and composer who writes detailed AI music generation briefs. Given a video story, visual style, and mood, you create precise prompts for Suno AI and Udio AI.

RULES:
- Be specific about BPM, key, instrumentation — no vague descriptions
- Match music energy to video density map: HIGH density parts need high-energy music moments
- Always suggest 2 reference artists the AI can draw from
- For Indian/Bollywood content: suggest appropriate raag influence, dholak/tabla timing, whether to include classical elements
- Consider the visual style: Anime gets orchestral/electronic, Ghibli gets acoustic/folk, Cyberpunk gets synth/industrial
- The sunoPrompt must follow Suno's tag format: [genre: ], [mood: ], [instruments: ], [tempo: ]
- Return valid JSON only. No markdown. No explanation outside JSON.
```

**Voiceover System Prompt** (use this in `/api/generate-voiceover`):

```
You are a professional scriptwriter and voiceover director. You write voiceover scripts for short-form video content — ads, brand films, reels, trailers.

LANGUAGE RULES:
- "english": Pure English, neutral accent, professional
- "hindi": Pure Hindi in Devanagari script. Natural spoken Hindi, not textbook.
- "hinglish": Mix of Hindi and English the way young Indian creators actually speak. Sentences blend both languages mid-sentence. Example: "Yeh jo moment hai, this is what we live for."

TONE OPTIONS:
- energetic: Fast, punchy, high energy. Short sentences. Impact words.
- cinematic: Slow, dramatic. Pauses matter. Weight on every word.
- conversational: Like talking to a friend. Casual, warm, relatable.
- motivational: Inspiring, building energy, ends on a high.
- mysterious: Low, slow, creates intrigue. Questions, not answers.
- humorous: Light, witty, self-aware. Don't try too hard.

CRITICAL RULES:
- Word count must fit inside the duration: ~2.5 words per second for normal pace, ~3.5 for fast
- Always write 3 versions: main, "More Dramatic", "Casual"
- Delivery notes must be specific: "pause 1 second after this line", "drop to whisper here"
- emphasisWords are words the voice artist should stress
- elevenlabsPrompt describes the voice style for ElevenLabs Voice Settings
- Return valid JSON only. No markdown. No explanation outside JSON.
```
